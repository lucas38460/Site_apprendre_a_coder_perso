Présentation de l'HTML du CSS et du Javascript

Par où commencer

Je voudrais commencer par expliquer le but de tout ça,
- Tout d'abord HTML n'est pas un langage de programation (nous ne pouvons pas faire bouger un robot avec ce langage) mais un langage de balisage "<>" soit un code simple à lire même sans compétence.

Le CSS ne sert pas non plus à programmer des robots mais à s'amuser esthetiquement sur une page internet.
Elle doit toujours être attachée à une page html, elle ne peut fonctionner sans. Différement à l' HTML, le css est un code en cascade et est aussi facile à écrire qu'à lire.

Le Javascript sert non seulement à faire du code sur une page, des animations... mais sert aussi énormément à faire plein d'autre choses comme faire des applis etc...
C'est un code très passionnant et très simple à utiliser dès qu'on a une logique compatible avec.
